# General Genre Video Game


## Build Instructions

### Dependencies 

Git

CMake

GCC

G++

SDL2:

- Ubuntu: `libsdl2-dev`

- Fedora: `SDL2-devel`

Lua

- Ubuntu: `liblua5.4-dev`

- Fedora: `lua-devel`

### Compilation

Run build commands:

`cmake CMakeLists.txt`

`make`

## Usage

Run `./StratagyGame`
